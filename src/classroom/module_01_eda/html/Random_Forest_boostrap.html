1. Qué se quiere mostrar

En un Random Forest no se construye un único árbol. Se construyen muchos árboles y cada uno aprende desde una muestra bootstrap distinta. Luego, cada árbol emite una predicción y el bosque combina esas predicciones.

Selección aleatoria de variables

En cada nodo, el árbol no evalúa todas las variables, sino un subconjunto aleatorio. Esto aumenta la diversidad entre árboles.

Impureza Gini

Para dividir un nodo, el árbol busca una separación que deje grupos más puros, es decir, con menos mezcla entre “Aprueba” y “No aprueba”.

Idea docente: Random Forest reduce la dependencia de un único árbol. En lugar de buscar una decisión perfecta, combina muchas decisiones parciales, construidas sobre datos y variables ligeramente diferentes.

8. Explicaciones para acompañar la simulación

Primero se toma una muestra bootstrap. Luego, en cada nodo, el árbol selecciona aleatoriamente algunas variables candidatas. Entre esas variables, busca la división que reduce mejor la impureza. El proceso se repite hasta llegar a una profundidad máxima o a una hoja.
La impureza Gini mide qué tan mezcladas están las clases dentro de un nodo. Si un nodo tiene solo estudiantes que aprueban, su impureza es baja. Si tiene una mezcla equilibrada de “Aprueba” y “No aprueba”, su impureza es mayor.
Porque cada árbol recibe una muestra bootstrap diferente y, además, en cada nodo evalúa un subconjunto aleatorio de variables. Esa diversidad es fundamental para que el bosque sea más robusto que un árbol individual.
Cada árbol vota una clase para el estudiante nuevo. Si la mayoría de los árboles vota “Aprueba”, el bosque predice “Aprueba”. Si la mayoría vota “No aprueba”, el bosque predice “No aprueba”.

9. Mini explicación para decir en clase

Random Forest construye muchos árboles de decisión. Cada árbol se entrena con una muestra bootstrap distinta, tomada con reemplazo desde el dataset original. Luego, en cada nodo, el árbol no evalúa todas las variables, sino un subconjunto aleatorio de variables candidatas. Entre esas variables candidatas, elige la división que mejora la pureza de los nodos, por ejemplo reduciendo la impureza Gini. Después, el estudiante nuevo recorre cada árbol y cada árbol emite un voto. La predicción final del bosque surge por votación mayoritaria. La clave no es construir un árbol perfecto, sino muchos árboles distintos y combinar sus decisiones.
