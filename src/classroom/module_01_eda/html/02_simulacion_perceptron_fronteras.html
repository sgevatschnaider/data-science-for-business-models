Controles

Lectura didáctica

La recta visible es el hiperplano de decisión de una neurona: w₁x₁ + w₂x₂ + b = 0

Los puntos de un lado se clasifican como 1 y los del otro como 0. En XOR, una sola recta no alcanza porque las clases positivas están diagonalmente enfrentadas.

Ejemplo avanzado

Cuando seleccionás XOR, la simulación dibuja dos fronteras auxiliares que representan una solución conceptual con capa oculta: primero se crean regiones intermedias, luego se combinan para resolver una frontera no lineal.
