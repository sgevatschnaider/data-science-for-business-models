1. Qué problema resuelve la regresión lineal

La idea

La regresión lineal busca explicar una variable numérica mediante una o más variables de entrada. En el caso más simple, tenemos una sola variable X y queremos predecir una salida Y.

ŷ = mX + b

La recta no pretende pasar por todos los puntos. Busca representar la tendencia general minimizando el error global.

Qué son m y b

m es la pendiente. Indica cuánto cambia la predicción cuando X aumenta una unidad.

b es la ordenada al origen o sesgo. Indica el valor de la predicción cuando X vale cero.

En lenguaje de redes: m es un peso y b es un bias. Entrenar es ajustar esos parámetros.

2. Función de pérdida: error cuadrático medio

Para saber si una recta ajusta bien, necesitamos medir el error entre el valor real y el valor predicho. Una medida habitual es el MSE, o error cuadrático medio.

MSE = (1/n) · Σ(yᵢ − ŷᵢ)²

El cuadrado hace que los errores positivos y negativos no se cancelen. También penaliza más los errores grandes.

3. Descenso por gradiente

Actualizar m y b

El modelo comienza con valores iniciales de m y b. Luego calcula el error y pregunta: ¿en qué dirección debo mover m y b para bajar el MSE?

m nuevo = m − α · ∂MSE/∂m b nuevo = b − α · ∂MSE/∂b

El parámetro α es el learning rate. Controla el tamaño del paso.

Gradientes de la regresión lineal

Para un modelo ŷ = mX + b, los gradientes se pueden calcular de forma directa:

∂MSE/∂m = (-2/n) · Σ xᵢ(yᵢ − ŷᵢ) ∂MSE/∂b = (-2/n) · Σ (yᵢ − ŷᵢ)

Estos valores indican cómo cambiar la pendiente y el bias para reducir la pérdida.

4. Entrenamiento y validación

Datos de entrenamiento

Son los puntos que el modelo usa para calcular los gradientes y ajustar los parámetros. La curva de entrenamiento muestra si el modelo aprende lo que ve.

Datos de validación

Son puntos separados que el modelo no usa para modificar m y b. Sirven para evaluar si la recta generaliza a datos no usados en el ajuste.

Idea clave: no alcanza con bajar el error de entrenamiento. Queremos que también baje o se mantenga bajo el error de validación.
