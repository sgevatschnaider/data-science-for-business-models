El machine learning, o aprendizaje automático, es una rama de la inteligencia artificial que permite construir modelos capaces de aprender patrones a partir de datos. A diferencia de la programación tradicional, donde una persona escribe reglas explícitas, en machine learning el modelo aprende esas reglas observando ejemplos.

Por ejemplo, en lugar de programar manualmente todas las condiciones para saber si un cliente comprará un producto, se le muestran al modelo muchos casos históricos de clientes que compraron y no compraron. A partir de esos datos, el algoritmo intenta encontrar patrones que le permitan hacer predicciones sobre nuevos clientes.

Random Forest pertenece a esta familia de métodos porque aprende relaciones entre variables de entrada y una variable objetivo. Su fortaleza está en que no depende de una única regla o de un único árbol de decisión, sino de muchos árboles que trabajan en conjunto.

El índice de Gini es una medida de impureza usada en árboles de clasificación. Sirve para saber qué tan mezcladas están las clases dentro de un nodo.

La intuición es simple: si un nodo contiene observaciones de muchas clases mezcladas, su Gini será alto. Si contiene observaciones de una sola clase, su Gini será cero.

Para dos clases, la fórmula puede pensarse así:

Gini = 1 - p₁² - p₂²

Donde p₁ y p₂ son las proporciones de cada clase.

Ejemplo:

Nodo A:
90% clase 1
10% clase 0

Nodo B:
50% clase 1
50% clase 0

El Nodo A tiene menor Gini porque está más cerca de ser puro. El Nodo B tiene mayor Gini porque está más mezclado.

En Random Forest, si usamos criterion="gini", cada árbol busca divisiones que reduzcan el índice de Gini.

La entropía es otra medida de impureza. Proviene de la teoría de la información y mide el grado de incertidumbre dentro de un nodo.

Cuando un nodo tiene una sola clase, no hay incertidumbre. La entropía es cero.

Cuando un nodo tiene clases muy mezcladas, la incertidumbre es alta. Por eso la entropía también es alta.

La fórmula general es:

Entropía = - Σ pᵢ log₂(pᵢ)

Donde pᵢ es la proporción de cada clase dentro del nodo.

La entropía suele ser un poco más sensible a cambios en las probabilidades que Gini, pero en la práctica ambos criterios suelen producir resultados similares.

Muchos árboles
Muestras distintas de datos
Subconjuntos aleatorios de variables

Random Forest y Boosting son métodos de ensamble, pero funcionan de manera distinta.

Random Forest construye muchos árboles independientes y luego combina sus resultados.

Boosting construye árboles en secuencia. Cada nuevo árbol intenta corregir errores del anterior.

Random Forest tiene varias ventajas importantes.

Primero, suele funcionar muy bien con datos tabulares. Segundo, no necesita tanto preprocesamiento como otros modelos. Tercero, puede capturar relaciones no lineales. Cuarto, reduce el sobreajuste de un único árbol. Quinto, permite estimar importancia de variables.

También es relativamente fácil de usar en la práctica. Muchas veces ofrece buenos resultados sin un tuning excesivo.

Por eso se lo considera un excelente modelo de referencia para problemas supervisados.

Random Forest también tiene limitaciones.

No es tan interpretable como un único árbol. Si el bosque tiene cientos de árboles, no podemos explicar fácilmente toda la lógica interna.

También puede ser pesado computacionalmente cuando hay muchos árboles, muchas variables o muchos datos.

Además, aunque funciona muy bien en muchos problemas tabulares, no siempre es el mejor modelo. En algunos casos, métodos de boosting o modelos especializados pueden superarlo.
