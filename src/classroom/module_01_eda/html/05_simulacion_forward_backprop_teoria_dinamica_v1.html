Forward propagation y Backpropagation

Esta simulación muestra el ciclo de aprendizaje de una red neuronal pequeña: primero calcula una predicción hacia adelante, luego mide la pérdida y finalmente ajusta los pesos hacia atrás para reducir el error.

1. Forward

Las entradas pasan por los pesos, se calculan las neuronas ocultas y se obtiene la predicción ŷ.

h = σ(Wx + b) ŷ = σ(Vh + c)

2. Pérdida

La red compara la predicción con el objetivo real. Si ŷ está lejos de y, la pérdida aumenta.

L = 1/2 · (ŷ − y)²

3. Backpropagation

El error se reparte hacia atrás para corregir las conexiones que más contribuyeron al error.

w nuevo = w − η · ∂L/∂w
