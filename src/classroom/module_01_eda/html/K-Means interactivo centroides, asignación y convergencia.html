K-Means interactivo: centroides, asignación y convergencia

Una simulación para explicar el algoritmo desde la intuición geométrica: los puntos se asignan al centroide más cercano, los centroides se recalculan como promedios y el proceso se repite hasta estabilizarse.

Inicio: datos sin etiquetas

Tenemos una nube de puntos sin clases conocidas. K-Means intentará formar grupos usando únicamente distancias y promedios.

Idea central: elegir K → inicializar centroides → asignar puntos → recalcular centroides → repetir.

En esta etapa conviene insistir en que el algoritmo no “sabe” qué grupo es correcto: solo busca una partición compacta alrededor de centros.
