Guía práctica: Autoencoders en Colab

Guía para implementar un autoencoder básico, un denoising autoencoder y un esquema de detección de anomalías. Está pensada para acompañar la clase con código ejecutable y preguntas de interpretación.

Objetivo: que el alumno no solo entrene una red, sino que interprete reconstrucciones, espacio latente, curvas de pérdida y umbrales.

Secuencia de trabajo

1. Preparación Imports y datos2. Autoencoder base Arquitectura3. Denoising Ruido controlado4. Anomalías Error y umbral5. Preguntas Evaluación conceptual

1. Preparación del entorno

import numpy as np
import matplotlib.pyplot as plt
import tensorflow as tf
from tensorflow.keras import layers, Model
from sklearn.model_selection import train_test_split

(x_train, _), (x_test, _) = tf.keras.datasets.mnist.load_data()
x_train = x_train.astype('float32') / 255.
x_test = x_test.astype('float32') / 255.
x_train = x_train.reshape((-1, 28*28))
x_test = x_test.reshape((-1, 28*28))

2. Autoencoder base

latent_dim = 32
inputs = layers.Input(shape=(784,))
x = layers.Dense(128, activation='relu')(inputs)
z = layers.Dense(latent_dim, activation='relu', name='bottleneck')(x)
x = layers.Dense(128, activation='relu')(z)
outputs = layers.Dense(784, activation='sigmoid')(x)

autoencoder = Model(inputs, outputs)
autoencoder.compile(optimizer='adam', loss='mse')
history = autoencoder.fit(x_train, x_train,
                          epochs=10,
                          batch_size=256,
                          validation_data=(x_test, x_test))
Discusión: cambiar latent_dim y observar la tensión entre compresión y calidad de reconstrucción.

3. Denoising autoencoder

noise_factor = 0.35
x_train_noisy = np.clip(x_train + noise_factor*np.random.normal(size=x_train.shape), 0., 1.)
x_test_noisy = np.clip(x_test + noise_factor*np.random.normal(size=x_test.shape), 0., 1.)

autoencoder.fit(x_train_noisy, x_train,
                epochs=10,
                batch_size=256,
                validation_data=(x_test_noisy, x_test))
recon = autoencoder.predict(x_test_noisy[:10])

4. Detección de anomalías

recon_all = autoencoder.predict(x_test)
errors = np.mean((x_test - recon_all)**2, axis=1)
threshold = np.percentile(errors, 95)
anomaly_flag = errors > threshold

plt.hist(errors, bins=50)
plt.axvline(threshold, linestyle='--')
plt.title('Distribución del error de reconstrucción')

El umbral no debe elegirse únicamente por conveniencia visual. Conviene validarlo con ejemplos reales, costos de error, proporción esperada de anomalías y conocimiento del dominio.
